_Escape Characters_ adalah karakter khusus pada string dalam Python, misalnya kita ingin memasukkan tanda petik sebagai elemen dalam string, 
maka kita harus menggunakan _escape characters_. _Escape characters_ diawali dengan tanda _backslash_ (\)

<Code
code = {
`teks = "Ia berkata, \\"Pergilah dari sini, ini bukan tempatmu!\\", seraya mengacungkan pistol"

print(teks) # output : Ia berkata, "Pergilah dari sini, ini bukan tempatmu!", seraya mengacungkan pistol`
}
lang="python"
/>

Beberapa _escape characters_ dalam Python:

|Kode|Output|
|---|---|
|`\'`|Petik satu (`'`)|
|`\"`|Petik dua (`"`)|
|`\\`|_Backslash_ (`\`)|
|`\n`|Baris baru (enter)|
|`\r`|Baris baru|
|`\t`|Tab|
|`\b`|_Backspace_|
|`\f`|_Form feed_|
|`\ooo`|Angka oktal (basis 8)|
|`\xhh`|Angka heksadesimal (basis 16)|
