Fungsi (_function_) adalah salah satu bagian paling penting dan paling sering digunakan dalam sebuah bahasa pemrograman
tingkat tinggi (_high-level programming language_). Fungsi memungkinkan kita untuk menggunakan sebuah formula secara berulang-ulang di dalam program kita. Bahkan,
fungsi ini dapat kita gunakan dalam program lain yang bisa jadi kita buat di waktu yang akan datang.

## _Syntax_ Dasar Fungsi

Untuk membuat sebuah fungsi, kita dapat menggunakan _keyword_ `def nama_fungsi()`, keyword `def` yang kita gunakan tersebut maskudnya adalah _define_, atau 
mendefinisikan fungsi yang kita buat.

<Code
lang="python"
theme="monokai"
code={
`def fungsi():
    print("Ini adalah sebuah fungsi")`
}
/>

## _Function Call_

_Function call_ adalah saat kita menggunakan (memanggil) sebuah fungsi, kita dapat melakukan _fuction call_ dengan _syntax_ `fungsi()`, baik saat kita 
menggunakan fungsi yang kita buat sendiri maupun fungsi bawaan / fungsi dari sebuah _library_.

<Code
lang="python"
theme="monokai"
code={
`import math

def fungsi():
    print("Ini adalah sebuah fungsi")
    
angka = math.sqrt(36)
print(angka) # output: 6
fungsi() # Output: Ini adalah sebuah fungsi`
}
/>


## `pass` Statement 

Ketika kita membuat sebuah fungsi, _statement_ di dalam fungsi tersebut tidak boleh kosong. Namun, apabila kita ingin membuat fungsi kosong, misalnya untuk 
me-_reserve_ nama fungsi agar tidak dipakai fungsi lain, kita dapat menggunakan statement `pass`.

<Code
lang="python"
theme="monokai"
code={
`def fungsi():
    # fungsi masih dalam proses pemikiran programmer
    pass`
}
/>

Dengan menggunakan `pass`, program akan tetap berjalan meskipun salah satu fungsi yang kita buat masih kosong.
